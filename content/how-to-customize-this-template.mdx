# How to Customize This Portfolio Template

Welcome to your new portfolio! This guide will walk you through customizing this template to showcase your unique work and personality.

## üìù Step 1: Update Your Personal Information

The main configuration file is located at `src/data/resume.tsx`. This is where you'll update most of your personal information.

### Basic Information

```typescript

  name: "Your Full Name",
  initials: "YN",
  url: "https://yourwebsite.com",
```

## üé® Step 2: Add Your Photo

1. Replace the file at `/public/pfp.jpeg` with your own photo
2. Make sure it's a square image (recommended: 400x400px or larger)
3. Supported formats: JPG, PNG, or WEBP

### Project Tips

- **Be specific**: Include metrics and results (e.g., "Improved performance by 40%")
- **Show impact**: Explain what problem your project solves
- **Add links**: GitHub repos, live demos, or case studies
- **Use keywords**: Include relevant technologies for SEO

## üé® Step 7: Customize Colors & Styling (Optional)

### Changing the Color Scheme

This template comes with a modern **blue accent color** by default, but you can easily customize it!

Edit `src/app/globals.css` to change colors:

```css
:root {
  --primary: 221.2 83.2% 53.3%;  /* This is the main accent color */
  --background: 0 0% 100%;        /* Background color */
  --foreground: 222.2 84% 4.9%;   /* Text color */
  /* ... other colors */
}
```

**Popular alternatives:**

- **Purple Theme**: `--primary: 262.1 83.3% 57.8%;`
- **Green Theme**: `--primary: 142.1 76.2% 36.3%;`
- **Orange Theme**: `--primary: 24.6 95% 53.1%;`
- **Pink Theme**: `--primary: 346.8 77.2% 49.8%;`

**Pro Tip**: Use [UI Colors](https://uicolors.app/) to generate a complete color palette!

### Fonts

## üö¢ Step 8: Deploy Your Portfolio

### Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Vercel will automatically detect Next.js and deploy!

### Custom Domain

1. Purchase a domain from a registrar
2. In Vercel dashboard, go to your project settings
3. Add your custom domain
4. Update DNS records as instructed

### Environment Variables

If you add any API keys or secrets:

1. Create a `.env.local` file (never commit this!)
2. Add the same variables in Vercel project settings
